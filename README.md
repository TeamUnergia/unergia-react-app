### Steps to run
* Clone the repository
``` bash
git clone https://github.com/TeamUnergia/unergia-react-app.git
```

* Go to folder
``` bash
cd unergia-react-app
```

* Install `node modules`
``` bash
npm install
```

* Run the project
``` bash
npm start
```
